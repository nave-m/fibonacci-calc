# このプログラムについて
フィボナッチ数 (http://ja.wikipedia.org/wiki/%E3%83%95%E3%82%A3%E3%83%9C%E3%83%8A%E3%83%83%E3%83%81%E6%95%B0)
を計算するプログラムです。

## 経緯
* 中途採用時にコードを書く面接を実施と聞く
* チーム内でこの先生きのこるにはこのぐらい書けないと
* ドヤ顔で社内SNSに公開し、先輩方に計算量やらフルボッコにされる
* 直して再アップロードして、またボコられての繰り返し

## 内容物
### test_fib.php
最初に書いたプログラム。再帰を使っており、計算量が膨大。

### test_fib2.php
再帰をやめ、計算機クラスにコンテクストを持たせた。

### test_fib3.php
計算機クラスのインタフェースとしてiteratorを実装。
単体テストもデータプロバイダを使うように変更

## テスト
phpunitをインストールした状態で、

    phpunit test_fib.php
